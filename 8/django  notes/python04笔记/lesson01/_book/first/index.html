Introduction

第一步

编写您的第一个Django应用程序

让我们通过例子学习。

在本教程中，我们将介绍创建基本的轮询应用程序。

它将由两部分组成：

  • 一个公众网站，让人们观看民意测验和投票。
  • 允许您添加，更改和删除轮询的管理站点。

我们假设你已经安装了Django。您可以通过在shell提示符（由$前缀指示）中运行以下命令来告诉Django已安装以及哪个版本：

$ python -m django --version
